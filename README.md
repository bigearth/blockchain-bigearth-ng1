# React Native port of blockchain.bigearth.io

## Intro

Built w/ 

* [Angular 1](https://angularjs.org/)
* [Twitter Bootstrap](FlexBox)

## Running locally

1. `git clone git@github.com:bigearth/blockchain-bigearth-ng1.git && cd blockchain-bigearth-ng1`
2. `npm install`
3. `npm start`
